SpreeRconomic
=============

Automatically transfer a shipped order to e-conomic.

Installation
------------

Add spree_rconomic to your Gemfile:

```ruby
gem 'spree_rconomic'
```

Configuration
-------------

layout ID
payment ID
product group
adjustments product number?
shipping product number?
customer primary key? phone? cvr?

Limitations
all is including VAT right now
